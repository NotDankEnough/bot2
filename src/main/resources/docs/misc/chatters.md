# Get chatters

The `!chatters` command allows you to get chatter list.

## Syntax

`!chatters`

## Responses

+ `Chatter list: https://paste.ilotterytea.kz/2ulUL`

## Important notes

+ Lists are sent to the third-party service ["ilotterytea's pastea"](https://paste.ilotterytea.kz).
